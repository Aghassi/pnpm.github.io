Instalação

Usando um script autônomo:

curl -L https://unpkg.com/@pnpm/self-installer | node

Via npm:

npm install -g pnpm

Depois de instalar o pnpm pela primeira vez, você pode atualizá-lo usando o pnpm:

pnpm install -g pnpm

Você quer usar o pnpm nos servidores de CI? Veja: Integração Contínua.
